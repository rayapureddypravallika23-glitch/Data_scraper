Data Scraper

Description

A Python program that extracts quotes from a website using web scraping techniques.

Features

- Fetches data from a website
- Extracts quotes using BeautifulSoup
- Displays scraped data

Technologies Used

- Python
- Requests
- BeautifulSoup

How to Run

1. Install required libraries.
2. Run the Python file.
3. View the extracted quotes.
